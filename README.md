<h1>Zone Based Firewall Configuration</h1>

 ### [YouTube Demonstration](https://youtu.)

<h2>Description</h2>
A Cisco Packet Tracer network security project that demonstrates secure router administrative access configuration (encrypted passwords, login banners, SSH access, and disabled Telnet) and the implementation of a Zone-Based Policy Firewall to control traffic between internal networks, DMZ servers, and the Internet.
<br />


<h2>Languages and Utilities Used</h2>

- <b>Cisco Packet Tracer</b> 
  

<h2>Environments Used </h2>

- <b>Windows 11</b>
